## Overview

 ezyExpress is a CLI tool for quickly scaffolding **Express.js projects**.  
It provides **Artisan-style commands** to generate controllers, models, routes, and middlewares with ease.  

- Fast project setup  
- Interactive prompts  
- Modular and extendable CLI
